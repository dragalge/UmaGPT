Guidelines:
1. Crop tightly around the digit only.
2. Use the same in-game font style and scale from the Held quantity area.
3. Prefer clean captures from debug images such as trackblazer_qty_held_value_<item>.png.
4. Keep a transparent or plain background if possible.

The bot will load these templates on startup and use template matching as an additional signal for quantity OCR.
